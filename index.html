Анимация на чистом js

Использование анимации на чистом html,css,js, без каких-либо библиотек.

HTML

<h2 class="simple__title animations">Простая анимация </h2> 
<p class="simple__text animations">class animations </p>

CSS

.simple__title{ 
    opacity: 0; 
    transform: translateY(-15px); 
    transition: 1.2s;
}  

.simple__title.anim{  
    transform: translateY(0);
    opacity: 1;  
} 

.simple__text{  
    transform: translateX(180px);
    transition: 1.2s;  
} 

.simple__text.anim{  
    transform: translateX(0); 
}
